🏨 VistaLodge – Hotel Booking Platform

VistaLodge is a full-stack hotel booking web application that allows users to explore hotels from around the world, book rooms, leave reviews, and create their own hotel listings after authentication. The platform is built using Node.js, Express, MongoDB Atlas, and EJS, with Cloudinary for image storage and Render for deployment.

✨ Features:

🔐 User Authentication (Login & Signup)
🌍 Browse hotels worldwide
🏨 Book rooms easily
⭐ Leave reviews and ratings
📝 Create your own hotel listings (after login)
🖼️ Image upload using Cloudinary
☁️ Deployed on Render

🛠️ Tech Stack:
Backend
Node.js
Express.js
MongoDB Atlas
Mongoose
EJS (Template Engine)

Services:
Cloudinary (Image Hosting)
Render (Deployment)
